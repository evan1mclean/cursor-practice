# TaskCountHeader Component

The TaskCountHeader component displays the number of tasks with proper pluralization.

## Overview

The TaskCountHeader component is a simple component that shows the total number of tasks. It automatically handles pluralization based on the count (e.g., "1 Task" vs "2 Tasks").

This component is typically used in conjunction with the DateHeader or as a standalone element to provide context about the number of tasks.

## Design Considerations

- The component uses a medium-sized font with a gray color to provide visual hierarchy
- The text automatically adjusts between singular and plural forms based on the count
- The component can be customized with additional classes for different styling needs
