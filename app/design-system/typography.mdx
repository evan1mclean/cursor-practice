# Typography System

This document outlines the typography system used in the Todo List application.

## Font Family

The application uses the Inter font family as the primary typeface, with system fonts as fallbacks:

```css
font-family: 'Inter', ui-sans-serif, system-ui, -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'Helvetica Neue', Arial, sans-serif;
```

## Font Sizes

### Heading Typography

- **Day of Week & Date Number (Thursday, 10th)**: `text-3xl font-semibold text-primary-dark`
  - Font size: 1.875rem (30px)
  - Line height: 2.25rem (36px)
  - Font weight: 600 (semibold)
  - Used for the main date header

- **Month (December)**: `text-xl font-normal text-gray-500`
  - Font size: 1.25rem (20px)
  - Line height: 1.75rem (28px)
  - Font weight: 400 (normal)
  - Used for the month in the date header

- **Task Count (12 Tasks)**: `text-lg font-medium text-gray-400`
  - Font size: 1.125rem (18px)
  - Line height: 1.75rem (28px)
  - Font weight: 500 (medium)
  - Used for displaying the number of tasks

### Body Typography

- **Task Text**: `text-base font-normal text-gray-800`
  - Font size: 1rem (16px)
  - Line height: 1.5rem (24px)
  - Font weight: 400 (normal)
  - Used for task descriptions

- **Completed Task Text**: `text-base font-normal text-gray-500 line-through`
  - Same as task text but with gray-500 color and line-through decoration
  - Used for completed tasks

### Supporting Typography

- **Time Stamps**: `text-sm font-medium text-gray-600`
  - Font size: 0.875rem (14px)
  - Line height: 1.25rem (20px)
  - Font weight: 500 (medium)
  - Used for time indicators (7:00 AM, 8:00 AM, etc.)

## Letter Spacing

- **Headers**: `tracking-tight` (-0.025em)
- **Body Text**: `tracking-normal` (0)
- **Supporting Text**: `tracking-normal` (0)

## Usage Guidelines

### Headers

- Use the heading typography for date headers and task counts
- Maintain proper hierarchy with the day/date being most prominent, followed by month and task count

### Task Text

- Use the body typography for task descriptions
- Apply line-through and gray-500 color for completed tasks

### Time Indicators

- Use the supporting typography for all time indicators
- Maintain consistent alignment on the right side of tasks

### Accessibility

- Maintain a minimum contrast ratio of 4.5:1 for all text
- Ensure text is resizable and readable at different screen sizes
- Use semantic HTML elements for proper document structure
