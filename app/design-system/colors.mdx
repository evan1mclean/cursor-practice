# Color System

This document outlines the color palette used in the Todo List application.

## Primary Colors

The primary colors are used for the background gradient and primary UI elements.

- **Light Purple (`primary.light`)**: `#8B5CF6` - Used for gradient start
- **Indigo (`primary.DEFAULT`)**: `#6366F1` - Main brand color, used in gradient middle
- **Dark Indigo (`primary.dark`)**: `#4F46E5` - Used for gradient end

## Secondary Colors

The secondary colors are used for action buttons and interactive elements.

- **Light Coral (`secondary.light`)**: `#F87171` - Used for hover states
- **Coral/Red (`secondary.DEFAULT`)**: `#EF4444` - Used for the add button and active states
- **Dark Red (`secondary.dark`)**: `#DC2626` - Used for pressed states

## Neutral Colors

Neutral colors are used for text, backgrounds, and UI elements.

- **White**: `#FFFFFF` - Used for card backgrounds and light text
- **Gray Scale**:
  - Gray 50: `#F9FAFB` - Subtle backgrounds, hover states
  - Gray 100: `#F3F4F6` - Light backgrounds, dividers
  - Gray 200: `#E5E7EB` - Borders, dividers
  - Gray 300: `#D1D5DB` - Disabled elements, secondary borders
  - Gray 400: `#9CA3AF` - Placeholder text
  - Gray 500: `#6B7280` - Secondary text, icons
  - Gray 600: `#4B5563` - Primary text for timestamps
  - Gray 700: `#374151` - Emphasized text
  - Gray 800: `#1F2937` - High emphasis text
  - Gray 900: `#111827` - Headings, very high emphasis

## Usage Guidelines

- **Background**: Use the primary gradient (`bg-gradient-primary`) for the app background
- **Cards**: Use white (`bg-white`) for card backgrounds
- **Primary Text**: Use Gray 800 for primary text content
- **Secondary Text**: Use Gray 500 for secondary text like timestamps
- **Action Buttons**: Use Secondary colors for action buttons like the add task button
- **Completed Tasks**: Use Gray 500 with strikethrough for completed task text
- **Dividers**: Use Gray 200 for dividers between tasks
